Класс Window - класс, который отвечает за основное окно. Наследуется от класса QMainWindow
Функции класса Window:
	-__init__ - функция-инициализатор, вызывает функцию unitUI. При первом запуске или удалении файла базы данных создаёт базу данных.

	-unitUI - функция отвечающая за интерфейс и назначения кнопок (основная)

	-excel_output - функция отвечающая за выгрузку данных, которые были внесены в базу данных, в excel таблицу

	-info - функция отвечающая за создание дополнительного окна объекта класса InfoWindow

	-add_outgo - функция отвечающая за ввод стандартных трат, создаёт объект класса Input_Window

	-add_plan - функция отвечающая за ввод планируемых трат, создаёт объект класса Input_Window

	-del_outgo - функция отвечающая за удаление стандартных трат, создаёт объект класса Del_Window

	-del_plan - функция отвечающая за удаление планируемых трат, создаёт объект класса Del_Window

	-return_only_outgo_graph - функция отвечающая за вывод графика по стандартным тратам. С помощью библиотеки mathplotlib 
		формирует график, который сохраняется как png изображение, которое выводится на экран пользователя. Создаёт окно 
		класса Output_Window.

	-return_only_plan_graph - функция отвечающая за вывод графика по планируемым тратам. С помощью библиотеки mathplotlib 
		формирует график, который сохраняется как png изображение, которое выводится на экран пользователя. Создаёт окно 
		класса Output_Window.

	-return_outgo_and_plan_graph - функция отвечающая за вывод графика по стандартным и планируемым тратам. С помощью 
		библиотеки mathplotlib формирует график, который сохраняется как png изображение, которое выводится на экран пользователя.

Класс Input_Window - класс, который отвечает за дополнительное окно, в котором осуществляется ввод. 
При инициализации принимает информацию о таблице. Наследуется от класса QWidget
Функции класса Input_Window:
	-__init__ - функция-инициализатор. Обозначает границы календаря и привязывает событие нажати на календарь к функции add_data

	-add_data - функция, которая добавляет введенную пользователем информацию в базу данных.

Класс Del_Window - класс, который отвечает за дополнительное окно, в котором осуществляется удаление информации. 
При инициализации принимает информацию о таблице. Наследуется от класса QWidget
Функции класса Del_Window:
	-__init__ - функция-инициализатор. Обозначает границы календаря и привязывает событие нажати на календарь к функции del_data

	-del_data - функция, которая отвечает за удаление информации на ту дату, которая была выбрана пользователем в календаре.

Класс InfoWindow - класс, который отвечает за вывод информации о программе. Наследуется от класса QWidget.
Функции класса InfoWindow:
	-__init__ - функция-инициализатор. В объект класса QPlainText выводит данные о программе из файла "Пояснительная_записка.txt"

Класс Output_Window - класс, который отвечает за вывод информации.
Функции класса Output_Window:
	-__init__ - функция-инициализатор. Отвечает за вывод png изображения графика 

В начале работы программа создаёт объект класса Window, показывает основное окно и продалжает работать, пока окна программы не закрыты
