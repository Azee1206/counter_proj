В программе существует пять типов окон:
	-Главное окно - главное окно, открыто всегда, в нём происходит навигация по остальным окнам с помощью кнопок:
		-Ввести обязательную/планируемую трату - открывает окно ввода
		-Удалить обязательную/планируемую трату - открывает окно удаления
		-Вывести график по обязательным тратам/планируемым тратам/по двум видам трат - открывает окно вывода
		-О программе - открывает окно с информацией о программе
		-Файл - позволяет перенести данные из программы в excel таблицу

	-Окно ввода - окно, в котором осуществляется ввод данных. В строку ввод вписывается трата, а после нажимается дата в календаре,
		на которую эта трата записывается. Бывает двух видов:
		-Ввод обязательных трат - окно, в котором осуществляется ввод данных о обязательных тратах.
		-Ввод планируемых трат - окно, в котором осуществляется ввод планируемых трат.

	-Окно удаления - окно, в котором происходит удаление данных. В календаре выбирается дата, на которую надо удалить данные.
		Бывает двух видов:
		-Удаление обязательных трат - окно, в котором осуществляется удаление данных о обязательных тратах.
		-Удаление планируемых трат - окно, в котором осуществляется удаление планируемых трат.

	-Окно вывода - окно, в котором выводится график на основе полученных данных. В нём можно ознакомиться с графиком, 
		который основан на информации о тратах и с самой информацией о тратах, которая отображается праее графика.
		 Бывает трёх видов, которые вызыываются нажатием на кнопки:
		-Вывести график по обязательным тратам - выводит график на основе данных по обязательным тратам.
		-Вывести график по планируемым тратам - выводит график на основе данных по планируемым тратам.
		-Вывести график по двум видам трат - выводит график на основе всех данных по тратам.
	
	-Окно информации - окно в котором выводится информация о программе
